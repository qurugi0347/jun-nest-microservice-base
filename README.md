## Description

NestJS Microservice 샘플 프로젝트

- Transport: Redis

## 기본 설정

### Docker

docker 폴더 내부 readme.txt 파일 따라서 도커 세팅

- Mysql
  - port: 33306
- Redis
  - port: 36379

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
